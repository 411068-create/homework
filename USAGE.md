# 中文成語卡片應用 - 使用說明

## 📚 應用介紹

這是一個純前端的中文成語學習應用，採用 HTML、CSS、JavaScript 開發，無任何框架依賴。用戶可以通過卡片翻轉來學習成語，並在管理頁面自定義成語內容。

## 🎯 主要功能

### 1. **成語卡片頁面** (index.html)
- 🃏 **卡片翻轉**：點擊卡片即可翻轉查看成語解釋
- ⬅️ ➡️ **上一張/下一張**：按鈕導航或使用鍵盤方向鍵
- 🔀 **隨機排序**：打亂成語順序進行練習
- 🔄 **重新開始**：返回第一張卡片

### 2. **管理頁面** (admin.html)
- ➕ **新增成語**：添加自訂成語和解釋
- ✏️ **編輯成語**：修改已有的成語解釋
- 🗑️ **刪除成語**：移除不需要的成語
- ✨ **自動填入解釋**：一鍵自動查詢成語解釋（需要網絡連接）

## 🚀 快速開始

### 1. 打開應用
在瀏覽器中打開 `index.html` 文件即可使用。

### 2. 學習模式
- **滑鼠操作**：點擊卡片翻轉，使用按鈕導航
- **鍵盤快捷鍵**：
  - `空格鍵`：翻轉卡片
  - `← 方向鍵`：上一張
  - `→ 方向鍵`：下一張

### 3. 管理成語
1. 點擊主頁面的 "管理成語" 按鈕
2. 輸入成語和解釋
3. 可選：點擊 "自動填入解釋" 自動查詢解釋
4. 點擊 "新增成語" 保存

## 📝 自動填入功能說明

### 工作原理
自動填入功能採用多層備份策略：

1. **優先級 1**：調用 API Ninjas 的成語 API
2. **優先級 2**：使用內置的本地成語數據庫（包含 100+ 常見成語）
3. **優先級 3**：嘗試調用其他中文 NLP API

### 支持的常見成語
應用內置了以下常見成語的解釋（無需網絡即可自動填入）：

- 臥虎藏龍、厚積薄發、十年磨劍
- 聞雞起舞、志在千里、千錘百煉
- 志同道合、腹有詩書氣自華、厚德載物
- 自強不息、寧靜致遠、人外有人
- ...以及更多 (詳見代碼中的 `getLocalIdiomDatabase()`)

### 使用步驟
1. 在成語輸入框中輸入成語名稱
2. 點擊 "✨ 自動填入解釋" 按鈕
3. 系統將自動查詢並填入解釋
4. 如果查詢失敗，可手動輸入解釋

## 💾 數據存儲

- **存儲方式**：使用瀏覽器的 LocalStorage
- **持久化**：關閉瀏覽器後數據仍然保留
- **跨標籤同步**：在一個標籤中新增的成語，其他標籤自動更新

### 核心技術
- **CSS 3D Transform**：實現卡片翻轉效果
- **LocalStorage API**：本地數據持久化
- **Fetch API**：調用遠程 API
- **事件監聽**：處理使用者交互和鍵盤快捷鍵
- **Responsive Design**：適配各種屏幕尺寸

## 🛠️ 自定義

### 修改預設成語
編輯 `script.js` 中的 `getDefaultIdioms()` 方法

### 添加更多本地成語
編輯 `admin.js` 中的 `getLocalIdiomDatabase()` 方法

### 更換 API 源
修改 `admin.js` 中的 `fetchIdiomMeaning()` 方法

## ⚠️ 注意事項

1. **網絡連接**：自動填入功能需要網絡連接
2. **瀏覽器支援**：需要支持 CSS 3D Transforms 和 LocalStorage 的瀏覽器
3. **隱私安全**：所有數據存儲在本地，不上傳服務器

## 🐛 常見問題

### Q: 卡片翻轉效果不工作
A: 確保使用現代瀏覽器（Chrome、Firefox、Safari、Edge）

### Q: 自動填入沒有反應
A: 檢查網絡連接，或手動輸入解釋

### Q: 數據丟失了
A: 檢查瀏覽器是否清除了 LocalStorage 數據

### Q: 怎樣匯出/導入數據
A: 目前需要手動複製 LocalStorage 中的 JSON 數據
